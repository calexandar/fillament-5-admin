# Roadmap

A SaaS platform built with Laravel, Filament, and Livewire.

## Tech Stack

- **Laravel 12** - PHP framework
- **Filament 5** - Admin panel
- **Livewire 4** - Reactive components
- **Tailwind CSS 4** - Styling

## Features

- User authentication
- Admin dashboard with Filament
- Real-time UI updates with Livewire
- Modern, responsive design

## Getting Started

```bash
composer install
npm install
npm run dev
php artisan migrate
php artisan serve
```
